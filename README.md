# Logan Thomas — Portfolio

Personal portfolio site for **Logan Thomas**, a mechanical engineering student at UCLA.
It highlights hands-on design, robotics, and fabrication work — mechanical design bridged
with electronics, controls, and applied research.

🔗 **Live site:** https://loganrthomas.github.io

## About this build

A lightweight, dependency-free site — plain HTML, CSS, and JavaScript with no framework
and no build step. Notable details:

- **Light / dark theme** that respects the OS preference and remembers the visitor's choice
- **Scroll-spy navigation** and reveal-on-scroll animations via `IntersectionObserver`
- **Accessible image lightbox** with full keyboard navigation
- **Responsive** from phone to desktop, plus a print stylesheet for clean PDF export
- **Performance-minded** — lazy-loaded images, fixed aspect ratios (no layout shift), semantic markup
